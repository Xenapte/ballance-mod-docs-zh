# WindowSizeFixer

Ballance 窗口大小问题修复器。

原版 Ballance Player 以窗口化模式启动时，窗口会比游戏实际分辨率稍微小一点（这是由于 Ballance 使用的窗口大小计算方式不适用于新版 Windows）。如果玩家不手动对窗口大小进行调节，游戏界面右下侧会被窗口裁断一部分，无法全部显示。本 Mod 会在游戏启动时自动修复此问题。

本 Mod 无需配置。
