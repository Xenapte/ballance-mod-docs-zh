# RealTimeMeter

现实时间计时器，用于计算竞速实际所用时间，可用于全关连打等竞速挑战中。

- 作者：BallanceBug
- 发布时间：2022 年 9 月

## 配置选项

- Main
  * Enabled - 是否启用。
  * StreakMode - 连打模式。设为 0 为禁用，否则效果为计时器通关后不重置，直到所设置的关卡数字号结束。
    + 例：将本选项设为 12，则玩家从第 1 关开始游戏，中间各关均连续不重置计时，直到通关 12 关为之。
