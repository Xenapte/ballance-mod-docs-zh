# SysInfoDisplay

游戏内叠加显示系统状态信息（现在时间、电量等）。

- 作者：BallanceBug
- 发布时间：2022 年 11 月

## 下载地址

- [地图下载站]
- [GitHub](https://github.com/Xenapte/BallanceSysInfoDisplay)

## 配置选项

- Main
  * Enabled - 是否启用本 Mod。
  * DisplayBatteryStatus - 是否显示电池状态。台式机等设备不存在电池供电选项，此时可以将此禁用。
- Text
  * BatterySameLine - 是否将电池数据和系统时间显示在同一行。
  * TextColor - 文字颜色，以 16 进制表示。
  * FontSize - 字号。
